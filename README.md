# TP Circos - Visualisation de données génomiques

## Objectif

Ce projet a pour objectif de représenter des données génomiques à l’aide du logiciel Circos.

## Étapes

### 1. Génération du karyotype

Extraction des chromosomes à partir des fichiers fasta.

### 2. Calcul de la densité de gènes

Calcul du nombre de gènes par fenêtres de 100 kb.

### 3. Analyse de la synténie

Identification des régions homologues entre chromosomes.

### 4. Visualisation

Création du graphique avec Circos.
